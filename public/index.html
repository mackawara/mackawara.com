Recent Projects

June 2023

AskMe

AskMe is a chatbot built on Node.js that provides users with educational, sport, business and religion-related information. The bot uses Redis for caching in memory to implement rate limiting functionality. Additionally, AskMe leverages the OpenAI API to provide more intelligent responses to user queries. The application has been deployed on an Amazon Web Services (AWS) virtual machine which ensures scalability and reliability of service delivery. Currently has 300 daily active users

September 2023

Farmers Club

During my time as a fullstack engineer at Livestock Wealth and as the Project Leader for "Farmers Club," I led a talented team in crafting an advanced platform tailored to empower farmers by providing an ecommerce platform for selling their produce. We utilized robust technologies including Node.js, TypeScript, MongoDB, WooCommerce integration for e-commerce capabilities, Redis caching to enhance performance and AWS Docker containers for deployment consistency. Additionally, we integrated the WhatsApp API to streamline communication. My responsibilities included overseeing system design and conducting thorough code reviews which ensured high-quality deliverables throughout our development process. One of my key contributions was leveraging Redis as an in-memory database solution that accelerated message processing times by 80%, significantly boosting user engagement on our platform. This project showcased not only my leadership skills but also demonstrated technical expertise and the ability to drive collaboration within a tech-driven environment,
